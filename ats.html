Claves Health — ATS 3.0

Configure o banco de dados gratuito (Firebase) para ativar o modo multi-usuário em tempo real. Leva menos de 3 minutos.

1
Acesse console.firebase.google.com com sua conta Google e clique em Criar projeto. Dê o nome claves-ats e desative o Google Analytics.
2
Dentro do projeto, clique em Firestore Database → Criar banco de dados → escolha Modo de teste → selecione região us-east1.
3
Clique no ícone </> (Web) para registrar o app. Copie o objeto firebaseConfig que aparecer.
4
Abra este arquivo HTML em um editor de texto (Bloco de Notas ou VS Code), localize o comentário ▼▼▼ COLE AQUI e substitua os valores de firebaseConfig pelos seus. Salve o arquivo.
5
Acesse app.netlify.com/drop, arraste e solte este arquivo HTML. Pronto — sua equipe acessa pelo link gerado!
Enquanto isso: o ATS já funciona normalmente nesta aba com armazenamento local. Os dados serão migrados automaticamente ao configurar o Firebase.
